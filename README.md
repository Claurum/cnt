# Информационная система "IT-ВУЗ"

Профессиональный одностраничный сайт для презентации современной информационной системы "IT-ВУЗ" с улучшенным дизайном и интерактивностью.

## Улучшения в проекте:

### 1. Визуальные улучшения:
- **Анимированный геометрический фон** с градиентами и плавающими элементами
- **Плавные переходы и анимации** для всех интерактивных элементов
- **Градиентные границы и тени** для карточек и секций
- **Интерактивные эффекты при наведении** на все элементы
- **Кастомный скроллбар** в стиле системы

### 2. Новый функционал:
- **Раздел с отзывами партнёров**
- **Анимированная статистика** с плавным подсчетом чисел
- **Улучшенная навигация** с активными состояниями при скролле
- **Красивые уведомления** при отправке формы
- **Ripple-эффекты** на кнопках

### 3. Технические улучшения:
- **Оптимизированная производительность** анимаций
- **Адаптивный дизайн** для всех устройств
- **Modern CSS** с использованием CSS Grid и Flexbox
- **ES6+ JavaScript** с асинхронными функциями

## Структура проекта:
- index.html # Главная страница
- styles.css # Основные стили
- icons.css # Стили для иконок
- script.js # JavaScript с интерактивными элементами
- README.md # Документация
- images/ # Папка с изображениями
