# Информационная система "IT-ВУЗ"

Проект одностраничного сайта для презентации информационной системы "IT-ВУЗ".

## Структура проекта:
- index.html - главная страница сайта
- styles.css - файл стилей
- script.js - JavaScript файл с интерактивными элементами
- images/ - папка с изображениями для сайта

## Функциональность сайта:
1. Адаптивный дизайн для всех устройств
2. Плавная прокрутка по якорным ссылкам
3. Интерактивная форма для заявки на демо-доступ
4. Анимации при прокрутке
5. Анимированная статистика
6. Мобильное меню

## Особенности системы "IT-ВУЗ":
- Интеллектуальное расписание
- Управление проектами и научной деятельностью
- Карьерный трекер с интеграцией IT-компаний
- Предиктивная аналитика успеваемости
- Цифровое портфолио студентов
- AI-ассистент для студентов

## Технологии разработки системы:
- Backend: Python/Django, PostgreSQL, Redis
- Frontend: React, JavaScript, CSS3
- Инфраструктура: Docker, Nginx, облачные платформы

## Запуск сайта:
Просто откройте файл index.html в любом современном браузере.

## Примечание:
Для корректного отображения иконок необходим доступ к интернету.
